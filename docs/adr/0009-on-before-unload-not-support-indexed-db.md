# 9. Unable to save data to indexedDB in onBeforeUnload state of web

Date: 2022-07-11

## Status

Accepted

## Context

- Unable to save data to indexedDB in onBeforeUnload state of web

## Decision

- Unable to save data to indexedDB in onBeforeUnload state of web

## Consequences

- Using session storage to save data in onBeforeUnload state of web
